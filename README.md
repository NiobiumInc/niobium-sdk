# Niobium FHE Tools Software Development Kit (SDK)

![SDK](https://github.com/NiobiumInc/niobium-sdk/actions/workflows/ci.yml/badge.svg) ![Pages](https://github.com/NiobiumInc/niobium-sdk/actions/workflows/pages.yml/badge.svg) ![Release](https://img.shields.io/github/v/release/NiobiumInc/niobium-sdk)

The Niobium Tools for FHE are documented, packaged, and released from this
repository.

The [Niobium Developer](https://developer.niobiummicrosystems.com) website
source for GitHub Pages is located here.

## Installation

See [Get the SDK](https://developer.niobiummicrosystems.com/sdk/).

## Usage

See the following documents to get started with the Niobium FHE SDK and
client/server runtime system:

* [Getting Started with the Niobium Client/Server RTS](https://developer.niobiummicrosystems.com/tutorials/getting-started-with-niobium-client-server-rts/).
* [Understanding the Niobium Client/Server RTS](https://developer.niobiummicrosystems.com/articles/understanding-niobium-client-server-runtime-tools/).
* [Understanding the Niobium FHE Software Development Kit](https://developer.niobiummicrosystems.com/articles/understanding-niobium-fhe-sdk/).

## Support

If you encounter problems using the Niobium FHE SDK, or would like to suggest
features or enhancements, please [open an issue on
GitHub](https://github.com/NiobiumInc/niobium-sdk/issues).
